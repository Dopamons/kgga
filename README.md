# KGGA 홈페이지 개편 목업 — 배포 세트

한국괌골프협회(KGGA) 홈페이지 개편안 목업입니다. 정적 HTML이라 GitHub Pages에 그대로 올리면 됩니다.

## 파일 구성
| 파일 | 페이지 | 비고 |
|---|---|---|
| `index.html` | 홈 (첫 화면) | 히어로·프로모션·골프장·가이드·협회소개·QnA, 실사진 포함 |
| `golf.html` | 골프장 상세 | 5개 코스 탭 전환, 스펙·편의시설·가로 스크롤 갤러리 |
| `hotels.html` | 골프 연계 호텔 | 12곳, PHR 계열(CCP 할인) 필터, 대표 사진 포함 |
| `weather.html` | 괌 날씨 | 라운딩 지수·오전/오후 브리핑·16일 예보·월별 시즌 |

모든 페이지 상단 메뉴로 서로 연결됩니다. 사진은 HTML에 내장(base64)되어 별도 이미지 폴더가 필요 없습니다.

## GitHub Pages 올리는 법
1. GitHub 저장소 만들기 (예: `kgga-mockup`) 또는 기존 `dopamons.github.io` 저장소의 하위 폴더(예: `/kgga`)에 이 폴더 안의 파일들을 업로드.
2. 저장소 **Settings → Pages** 에서 Source를 `main` 브랜치로 지정하고 저장.
3. 잠시 후 아래 주소로 접속:
   - 새 저장소: `https://<아이디>.github.io/kgga-mockup/`
   - 기존 사이트 하위폴더: `https://<아이디>.github.io/kgga/`

`index.html`이 자동으로 첫 화면이 됩니다.

## 나중에 교체할 것 (실서버 반영 전)
- 실촬영 고화질 사진 (히어로·골프장·호텔). 현재는 가이드북 PDF에서 추출한 사진.
- 프로모션 3장 가격·예약 링크 파라미터(`/guamfree/golf/?strGolf=...&intSeq=...`) 실판매 기준 확정.
- 푸터 사업자 정보(대표·사업자번호·주소·전화) 실제 값.
- 웨스틴·하얏트·크라운 호텔 사진은 PDF 해상도가 낮아 공식 고화질로 교체 권장.
